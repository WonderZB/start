Step 3 of 3 — Cross-device sync

Connect Google sync

Firebase (free, by Google) syncs history across all devices. Skip for local storage only.

  1. console.firebase.google.com → Add project
  2. Authentication → Get started → Enable Google
  3. Add your GitHub Pages domain under Authorized domains
  4. Firestore Database → Create → Production mode
  5. Firestore → Rules → replace all with:
    rules_version = '2';
    service cloud.firestore {
     match /databases/{database}/documents {
      match /users/{uid}/{d=**} {
       allow read,write: if request.auth.uid==uid;
      }
     }
    }
  6. ⚙ Project settings → Web app → register → copy the config JSON
