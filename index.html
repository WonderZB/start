Step 1 of 3 — AI engine

Get your Gemini key

Free tier: 1,500 requests/day. No credit card needed.

  1. Go to aistudio.google.com/apikey
  2. Click Get API key → Create API key
  3. Copy the key (starts with AIza) and paste below
Step 2 of 3 — Web search

Brave Search API

Shows real search results embedded in WonderStart — no redirects. Free: 2,000 searches/month. Skip to use DuckDuckGo redirect instead.

  1. Go to api.search.brave.com/register
  2. Create a free account and verify your email
  3. Go to API Keys → create a key
  4. Paste the key below
Step 3 of 3 — Cross-device sync

Connect Google sync

Uses Firebase (free, by Google) to sync history across all your devices. Skip to use local storage only.

  1. Go to console.firebase.google.com → Add project
  2. Authentication → Get started → Enable Google
  3. Add your GitHub Pages domain under Authorized domains
  4. Firestore Database → Create database → Production mode
  5. Firestore → Rules → replace all with:
    rules_version = '2';
    service cloud.firestore {
     match /databases/{database}/documents {
      match /users/{uid}/{d=**} {
       allow read,write: if request.auth.uid==uid;
      }
     }
    }
  6. ⚙ Project settings → Web app → register → copy the config JSON
